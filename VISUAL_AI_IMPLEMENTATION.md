# 拾光 v1.0 Visual AI 实施说明

## 已实现的可运行内容

### 首页与品牌体验
- 首屏采用高端品牌感 + 情感冲击感。
- 桌面端左文右图，移动端上下结构。
- 首屏使用指定主文案与产品解释副标题。
- 官方主 Demo 改为梦幻星空情侣故事，亲情胶片案例保留为次级案例。
- 价格放在样片、AI 推荐和产品价值之后。
- 未加入虚构用户评价。

### 官方主 Demo
路由：`/demo/galaxy`

- 《距离之外，我们仍在同一片星空》完整演示内容。
- 陈默 / 林晚真实感虚构人物设定。
- 解锁问题、演示答案与两次错误提示。
- 90 秒场景时长配置。
- 沉浸播放 / 自由浏览。
- 暂停、上一幕、下一幕与声音开关。
- 第 1095 号星、三周年情书、未来约定。
- 心情选择 + 文字回复，本次体验内展示，不写数据库。
- 结束后可沿用梦幻星空或让 AI 重新推荐。

### AI 主题推荐
路由：`/create`

- 桌面端左侧逐题问卷、右侧实时氛围预览。
- 移动端单题推进。
- 关系、纪念场景、最多三项情绪、真实小事。
- “使用演示故事”自动填写陈默与林晚答案。
- 3 秒分析状态。
- 最佳主题 + 一个备选主题。
- 不显示伪精确百分比，只使用“非常适合 / 比较适合”。
- 备选主题 10 秒微体验。
- 未填写真实故事时允许继续，但不自动编造经历。

API：`POST /api/ai/theme-recommend`
- 配置 DeepSeek 后使用真实模型。
- 未配置或请求失败时使用稳定规则引擎。
- JSON Output 结构化返回。

### 独立套餐推荐
路由：`/plan-recommend?theme=galaxy`

- 主题确定后再询问视频、多人共创、保存期限和 AI 修改需求。
- 推荐轻定制 / 深度定制 / 私人策划。
- 仍允许用户自行选择其他套餐。
- 显示保存增值费用和预计合计，不在付款末尾突然出现。

### 付款前一次真实 AI 预览
路由：`/story-preview?theme=galaxy&plan=deep`

- 每个草稿生成一次封面标题、短副标题与一小段故事示例。
- 使用浏览器草稿标识实现同一草稿幂等返回。
- 配置 Supabase 后使用 `ai_preview_drafts` 持久控制一次预览。
- 未配置 Supabase 时使用进程内控制，仅适合本地演示。
- 预览后提供“保存这份故事并继续”和“修改我的答案”。

API：`POST /api/ai/preview`

### 三套独立制作台
- `/studio/theme/film`
- `/studio/theme/galaxy`
- `/studio/theme/cinema`

#### 温暖胶片
- 素材抽屉 + 打开相册 + 收件人预览。
- 左右翻页与底部缩略图。
- 受控拼贴视觉。
- 快捷编辑浮层与正文直接编辑。
- 阳光光斑、纸张、拍立得与手写日期。

#### 梦幻星空
- 中央自由星座画布。
- 星点拖动、选中、故事编辑与连接模式。
- 可视化连线与自动整理入口。
- 当前主星、并行任务与收件人预览。

#### 私人电影
- 左侧分镜卡片、中间电影预览、右侧镜头设置。
- 上方分镜卡片 + 底部连续胶片进度条。
- 默认简化时间线，支持展开专业多轨时间线。
- 镜头时长、字幕、旁白与转场入口。

### 三个保留核心能力
三套制作台均包含：
- 顶部简化自动保存状态。
- 通知中心与页面最重要通知摘要。
- 整份礼物版本入口和单段版本概念。
- 主题切换预览入口。
- “减少动态效果”开关。
- 主题专属主名称 + “收件人视角模拟”副标题。
- 基础检查 + 慢速网络、错误解锁、媒体失败、开放时间限制入口。

## 数据库迁移

执行：

```sql
supabase/migrations/010_visual_ai_preview.sql
```

该迁移新增付款前 AI 预览草稿表，用于在生产环境可靠限制每个草稿一次预览。

## DeepSeek 环境变量

```env
DEEPSEEK_API_KEY=你的密钥
DEEPSEEK_MODEL=deepseek-v4-flash
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

代码采用 JSON Output，并在提示词中明确要求 JSON。文案场景关闭思考模式，降低延迟。

## 当前实施边界

本版本已经完成可运行的视觉、交互与 DeepSeek 接口基础，但以下部分仍属于后续生产接线，而不是本次虚假声称已完成：

1. 三套新制作台目前使用浏览器本地演示数据，尚未替换原 `/studio` 的订单与 Supabase 卡片持久层。
2. 官方 Demo 使用完整文案和 CSS 视觉占位，尚未加入最终统一人物照片、正式授权音乐与真人语音素材。
3. 纪念档案离线网页、PDF、素材 ZIP 的后台生成 Worker、授权审批和七天下载任务尚未实现。
4. 通知、版本和发布模拟在新主题制作台中已完成界面与交互原型，生产数据库表和任务队列仍需下一阶段接入。
5. 人工付款流程沿用原项目；AI 预览结束后进入现有套餐/付款流程，尚未重构为新的“草稿转正式项目”事务。
