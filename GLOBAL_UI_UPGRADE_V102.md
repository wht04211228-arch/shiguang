# 拾光 v1.0.2-beta.3 全局交互与 UI 升级说明

## 本次修正的核心问题

上一版三大主题制作台中存在一批“只有视觉、没有行为”的按钮。通过 TypeScript AST 全局审计，旧版共发现 31 个无行为按钮，集中在：

- 通知中心与版本历史
- 收件人视角模拟
- 温暖胶片素材与故事创建
- 梦幻星空缩放、连线、自动整理与 AI 入口
- 私人电影新增分镜、转场、旁白、AI 字幕和节奏整理

v1.0.2-beta.3 已将这些按钮全部接入可见操作、编辑面板、状态反馈或明确跳转，并增加自动审计脚本防止回归。

## 首页信息架构重构

首页在首屏与案例之间增加三个核心区域：

1. **拾光是什么**：明确说明它不是普通电子贺卡或单纯视频，而是可打开、聆听、探索和回应的私人数字礼物。
2. **五步制作引导**：从“送给谁”到“发布并等待回应”，每一步可以点击查看结果和下一步。
3. **三套制作台说明**：清晰展示相册整理桌、专属星座画布和私人电影剪辑台分别适合什么人、能做什么。

顶部导航新增“它是什么”“怎么制作”，并新增独立 `/help` 使用帮助页。

## 三大主题制作台升级

### C 温暖胶片

- 素材抽屉展示可点击的照片、回忆、信件、票根、语音和朋友祝福。
- 新增三步故事创建面板：选择故事类型 → 填写真实细节 → 确认主题映射。
- 照片快捷编辑打开真实设置面板。
- 手写日期、语音、素材和版本操作均有反馈。

### B 梦幻星空

- 星点支持拖动、选择和连接。
- 缩放、回到完整星座、自动整理全部可用。
- 照片声音上传与 AI 调整打开可操作面板。
- 收件人模拟支持设备、声音、浏览模式和四类异常测试。

### A 私人电影

- “新增一幕”会真实加入分镜与时间线。
- 转场、旁白、AI 字幕、片尾署名均进入可编辑面板。
- 自动整理节奏会更新镜头时长。
- 简化时间线与专业轨道保持可切换。

## 全局统一

- 案例页和订单页改用统一主题感知导航。
- 修复所有空链接与旧样片地址。
- Dashboard、登录和订单中的演示制作入口统一指向新版主题制作台。
- 所有动作提供抽屉、Toast、状态变化或页面跳转，不再静默无反应。

## 新增质量门禁

运行：

```powershell
npm run audit:ui
```

会扫描 `app` 和 `components` 下所有 TSX/JSX 文件，并阻止以下问题进入构建：

- 没有 `onClick`、`formAction` 或提交行为的按钮
- `href=""` 或 `href="#"` 的空链接

完整检查：

```powershell
npm run check
```

执行顺序为：

```text
UI 交互审计 → TypeScript 类型检查 → Next.js 生产构建
```

## 部署

```powershell

git add -A
```
