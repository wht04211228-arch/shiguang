- [ ] 新项目完整执行 `supabase/schema.sql`

## 初稿与修改

- [ ] 制作台发布后订单仍为制作中，而不是已交付

- [ ] 上一轮仍待确认时不能重复发起
- [ ] 客户可以打开预览并批准
- [ ] 批准后订单进入 `fulfilled / delivered`
- [ ] 客户可以提交至少 8 个字的修改意见
- [ ] 修改后 `revision_count` 增加
- [ ] 超过套餐修改次数后不能继续提交
- [ ] 新一轮初稿能够正常创建

## 制作任务

- [ ] 正式提交问卷后自动生成三条基础任务
- [ ] 重复提交问卷不会重复生成任务
- [ ] 管理员可以添加任务
- [ ] 可以设置负责人和截止时间
- [ ] 任务状态刷新后仍然保留
- [ ] 逾期任务在后台指标中出现

## 自动提醒

- [ ] 配置 `OPERATIONS_NOTIFICATION_EMAIL`
- [ ] 配置高强度随机 `CRON_SECRET`
- [ ] 未带授权头访问提醒接口返回 401
- [ ] Vercel Cron 每天触发；付费计划如改为更高频率，应重新验证限额
- [ ] 未提交问卷超过 24 小时会提醒用户
- [ ] 初稿超过 24 小时未确认会提醒用户
- [ ] 距交付不足 24 小时会提醒运营
- [ ] 同一订单、提醒类型、收件人和日期的任务只能成功抢占一次
- [ ] Resend 未配置时只写日志，不阻断业务

## 评价与案例授权

- [ ] 只有已交付订单能提交评价

- [ ] 媒体授权仍要求对具体素材再次确认
