## 付款与权益

- [ ] 人工付款凭证可上传、驳回和重新提交
- [ ] 管理员确认真实到账后才开放制作权限
- [ ] 邀请人数与保存期限计入应付金额
- [ ] 补差价订单只能向上升级

- [ ] 未付款订单不能上传云端素材、调用 AI 或发布

## 多人共创

- [ ] 公共邀请链接可投稿
- [ ] 专属邀请可独立撤销和延长期限
- [ ] 免注册投稿可获得编辑凭证
- [ ] 登录参与者可绑定账号
- [ ] 每位参与者最多3张照片和1段语音
- [ ] 视频数量与时长按套餐限制
- [ ] 人数达到档位上限后停止新增投稿
- [ ] 购买者可确认、排序、隐藏、删除和退回修改
- [ ] 购买者不能修改投稿者原文
- [ ] 匿名投稿对收件人隐藏身份、对购买者可见
- [ ] 发布前锁定版本后停止编辑

## 收件人共同纪念空间

- [ ] 正确解锁后才能添加新内容
- [ ] 可上传照片与声音并保存新回忆
- [ ] 收件人账号绑定成功
- [ ] 绑定后进入共同管理阶段
- [ ] 收件人邀请申请可由购买者批准或拒绝
- [ ] 批准后收件人可在剩余额度内生成邀请
- [ ] 隐私举报可进入管理员复核队列

## 安全

- [ ] 未解锁时不返回正文、共创和收件人内容
- [ ] 未到开放时间不返回正文
- [ ] 失效后读取、回复和新增纪念内容均被拒绝
- [ ] 文件类型、体积、数量和视频时长校验有效
- [ ] 解锁 Cookie 为 HttpOnly、Secure（生产）和 SameSite=Lax
- [ ] Secret Key 没有进入 GitHub 或浏览器包
- [ ] 普通用户不能进入 `/admin`、`/admin/payments`、`/admin/reports`
