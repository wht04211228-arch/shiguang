# 拾光 v0.9.4 构建与验收报告

## 核心改动

- 固定十步制作导航替换为动态制作旅程；
- 根据套餐、视频权益、共创权益和内容状态自动生成任务；
- 增加 1 个主推荐和最多 2 个并行任务；
- 增加完整旅程弹层；
- 增加全部功能入口；
- 发布后自动切换为交付与回应流程；
- 共创管理器实时同步投稿统计给动态旅程；
- 增加收件人视角模拟完成记录；
- 制作台底部主按钮自动跟随当前推荐任务。
