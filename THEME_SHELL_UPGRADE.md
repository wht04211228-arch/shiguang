# 拾光 v1.0.1-beta.2 主题壳层升级

本次升级解决“内容区已经切换主题，但顶部导航仍保持旧中性样式”的割裂问题。

## 已完成

- 公共导航由固定白色栏替换为全宽 Theme-aware Header。
- 温暖胶片、梦幻星空、私人电影拥有独立导航材质、配色、按钮和环境细节。
- 导航结构保持一致，切换主题不需要重新学习。
- AI 主题推荐页会跟随当前实时预览主题同步切换页面壳层与导航。
- 套餐推荐页、一次 AI 故事预览页直接继承已确认主题。
- 三套主题制作台顶部品牌栏与当前工作区统一。
- 梦幻星空官方 Demo 导航升级为星空主题，并补充品牌标识与退出入口。
- 桌面端导航改为全宽背景 + 居中内容容器，解决宽屏两侧出现不协调色块的问题。
- 移动端新增可展开导航菜单，不再简单隐藏全部导航项。

## 主题表现

### C 温暖胶片

奶油纸张、复古米棕、轻纸张纹理、暖棕主按钮。

### B 梦幻星空

深空靛蓝渐变、淡紫星光、微弱星轨、淡紫主按钮。

### A 私人电影

电影黑蓝、暖金分隔线、胶片式细节、暖金主按钮。

## 覆盖升级

本补丁基于 `v1.0.0-beta.1`。将补丁内容复制到当前项目根目录覆盖，然后执行：

```powershell
npm install
npm run typecheck
npm run build
```

本次升级不需要新增 Supabase 迁移，也不需要新增环境变量。
